# Introduction

This is a collection of functions that you can use for visalization of the raw *mzXML* files, independently from their acquisition mode (i.e. DIA vs DDA). This collections supports visualization of raw MS1, MS2, multiple scans, extracted ion chromatograms (XIC/BIC), as well as full chromatograms. 

# MS1 spectra 

## `get_scan_MS1`

The `get_scan_MS1` function is used to extract and visualize the **MS1 spectrum** for a specified scan number. This spectrum displays the mass-to-charge ratio ($m/z$) against the corresponding ion intensity.

### Args
* chrom_data{:Dict}: A dictionary containing the full chromatographic data. The chromatographic data structure, typically generated by MS_Import.jl, which contains the MS1 data ("MS1" key).
* scan_n{:Int}: The scan number from which the MS1 spectrum should be extracted.
* window{:Tuple of float64} - optional: Specifies an $m/z$ range (min_mz, max_mz) to zoom the plot view. If provided, peak annotation is disabled.
* n_p{:Int} - optional: The number of top peaks to annotate with their $m/z$ values. Defaults to 3. If set to 0, no peaks are annotated.
* offset{:Float64} - optional: A parameter for the internal sig_select utility function, used to label significant peaks. Defaults to 0.1.

### Output

* f: A figure object containing the generated MS1 spectrum plot.

### Usage

```julia

# Extract the spectrum at MS1 scan 600
scan_n = 600
f = get_scan_MS1(chrom_data, scan_n)

# Changing the number of labeled points

n_p = 5

f = get_scan_MS1(chrom_data, scan_n; n_p = 5)

# Zooming into an m/z window

window = (200, 400)

get_scan_MS1(chrom_data, scan_n; window = (200, 400))

``` 

## `get_rt_MS1`

The `get_rt_MS1` function is used to extract and visualize the MS1 spectrum that corresponds to the closest available scan at a specified retention time (RT). This spectrum displays the mass-to-charge ratio ($m/z$) against the corresponding ion intensity.

### Args
* chrom_data{:Dict}: A dictionary containing the full chromatographic data. The chromatographic data structure, typically generated by MS_Import.jl, which contains the MS1 data ("MS1" key).
* rt_value{:Float64}: The target retention time (in minutes) from which the MS1 spectrum should be extracted. The function will find the scan with the closest RT.
* window{:Tuple of float64} - optional: Specifies an $m/z$ range (min_mz, max_mz) to zoom the plot view. If provided, peak annotation is disabled.
* n_p{:Int} - optional: The number of top peaks to annotate with their $m/z$ values. Defaults to 3. If set to 0, no peaks are annotated.
* offset{:Float64} - optional: A parameter for the internal sig_select utility function, used to label significant peaks. Defaults to 0.1.

### Output

* f: A figure object containing the generated MS1 spectrum plot.

### Usage 

```julia 

# Extract the spectrum closest to 5.0 minutes
rt_value = 5.0 

f = get_rt_MS1(chrom_data, rt_value)

# Extracting the spectrum closest to 5.0 minutes and annotating 5 peaks

rt_value = 5.0
n_p = 5

f = get_rt_MS1(chrom_data, rt_value; n_p = n_p)

# Zooming into a specific m/z window (200 to 400) at the m/z window

rt_value = 5.0
window = (200, 400)

get_rt_MS1(chrom_data, rt_value; window = window)

```

## `get_multi_scanMS1`

The `get_multi_scanMS1` function is a powerful utility for mass spectrometry data visualization, designed to plot multiple MS1 spectra from different time points (scans) in a single figure. Each spectrum is displayed in its own subplot, showing ion intensity versus mass-to-charge ratio ($m/z$).

### Args
* chrom_data{:Dict}: The required dictionary containing the full chromatographic data, specifically the MS1 data (under the "MS1" key).
* scans_n{:List of Int}: The required list of specific scan numbers whose MS1 spectra will be extracted and plotted.
* window{:Tuple of float64}: Optional. Specifies an $m/z$ range (min_mz, max_mz) to zoom the plot view. If provided, peak annotation is disabled.
* n_p{:Int}: Optional. The number of top peaks to annotate with their $m/z$ values. Defaults to 3. If set to 0, no peaks are annotated.
* offset{:Float64}: Optional. A parameter (default: 0.1) used internally to select significant peaks for annotation.

### Output

* f: A figure object containing the generated MS1 spectrum plot.

### Usage 

```julia 

# Plotting three scans in a file
scans_n = [100, 200, 300]
f = get_multi_scanMS1(chrom_data, scans_n)

# Changing the number of points to be labeled

n_p = 5 
f = get_multi_scanMS1(chrom_data, scans_n; n_p = n_p)

```

## `get_multi_rtMS1`

The `get_multi_scanMS1` function is a powerful utility for mass spectrometry data visualization, designed to plot multiple MS1 spectra from different retention times in a single figure. Each spectrum is displayed in its own subplot, showing ion intensity versus mass-to-charge ratio ($m/z$).

### Args
* chrom_data{:Dict}: The required dictionary containing the full chromatographic data, specifically the MS1 data (under the "MS1" key).
* rt_values{:List of Float}: The required list of specific retention times whose MS1 spectra will be extracted and plotted.
* window{:Tuple of float64}: Optional. Specifies an $m/z$ range (min_mz, max_mz) to zoom the plot view. If provided, peak annotation is disabled.
* n_p{:Int}: Optional. The number of top peaks to annotate with their $m/z$ values. Defaults to 3. If set to 0, no peaks are annotated.
* offset{:Float64}: Optional. A parameter (default: 0.1) used internally to select significant peaks for annotation.

### Output

* f: A figure object containing the generated MS1 spectrum plot.

### Usage 

```julia 

# Plotting three rt values in a file
rt_values = [5.0, 10.0, 15.0]
f = get_multi_rtMS1(chrom_data, rt_values)

# Changing the number of points to be labeled

n_p = 5 
f = get_multi_rtMS1(chrom_data, rt_values; n_p = n_p)

# Zooming into an m/z window 

window = (200, 400)
get_multi_rtMS1(chrom_data, rt_values; window = window)

```
